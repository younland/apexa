## 项目介绍

Apexa 是一款极简、以离线为先的 API 测试工具，旨在追求速度和峰值性能。

## ✨ 特色

- ✅ 100% 离线——测试无网络依赖的 API
- ⚡ 超轻量化——体积最小，启动快速
- 🎯 精确请求/响应检查——查看头部、正文、状态、时序
